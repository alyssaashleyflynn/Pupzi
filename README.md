# Pupzi Prototype

A working Vite + React prototype for the pixel-style dog care app.

## Run locally

```bash
npm install
npm run dev
```

## Upload to GitHub

1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Commit changes.

## Deploy to Vercel

1. Go to Vercel.
2. New Project.
3. Import your GitHub repository.
4. Framework preset: Vite.
5. Build command: `npm run build`
6. Output directory: `dist`
7. Deploy.
